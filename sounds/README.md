# Archivos de Sonido Necesarios

Para que el juego funcione con todos los efectos de sonido, necesitas agregar los siguientes archivos MP3 a esta carpeta:

## Archivos requeridos:

| Archivo | Descripción | Duración recomendada |
|---------|-------------|---------------------|
| `correct.mp3` | Sonido de respuesta correcta | 1-2 segundos |
| `wrong.mp3` | Sonido de respuesta incorrecta | 1-2 segundos |
| `tick.mp3` | Tick del temporizador | 0.5 segundos |
| `suspense.mp3` | Música de suspense (loop) | 10-30 segundos |
| `win.mp3` | Fanfarria de victoria | 3-5 segundos |
| `lifeline.mp3` | Sonido al usar comodín | 1-2 segundos |
| `select.mp3` | Sonido al seleccionar opción | 0.5 segundos |

## Dónde obtener sonidos gratuitos:

1. **Freesound.org** - https://freesound.org
2. **Pixabay** - https://pixabay.com/sound-effects
3. **Mixkit** - https://mixkit.co/free-sound-effects

## Nota:
El juego funcionará sin estos archivos, simplemente no tendrá efectos de sonido.
